# 🎯 Quizify - MERN Quiz App

🚀 A full-stack quiz application built using the MERN stack (MongoDB, Express, React, Node.js).

---

## ✨ Features

✅ User Authentication (Login / Signup)  
✅ Multiple Quizzes  
✅ Score Tracking  
✅ Leaderboard 🏆  
✅ Dashboard Analytics 📊  
✅ Beautiful UI (Glassmorphism Design)  
✅ Search & Filter Quizzes  
✅ Performance Tracking  

---

## 🛠 Tech Stack

**Frontend:**
- React.js
- CSS (Glass UI)

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB
